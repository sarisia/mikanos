# MikanOS

# Build

```
$ source init.sh
$ make
```

# Run QEMU

```
$ source init.sh
$ make run
```

# Run QEMU with GDB debug enabled

```
$ source init.sh
$ make debug
```

# Debugging with VSCode cppdbg

```
$ source init.sh
$ make debug
```

then, run `(gdb) Attach to QEMU` from VSCode debug menu
